# Word文档专业化格式化操作指南

## 目标
将《软件设计说明》文档调整为专业、官方、严肃、统一的格式，符合软件设计说明规范。

---

## 一、字体设置详解

### 1. 正文格式设置

**步骤：**

1. **选中所有正文文本**（Ctrl+A全选，然后按住Ctrl点击标题取消选中）
2. **设置字体**：
   - 右键 → 字体
   - 中文字体：仿宋GB2312
   - 字号：三号（16磅）
   
3. **设置段落格式**：
   - 右键 → 段落
   - 行距：固定值 29磅
   - 首行缩进：2字符
   
4. **批量应用**：
   - 可以先设置一段正文
   - 然后双击格式刷，批量刷其他正文段落

### 2. 一级标题设置

**步骤：**

1. **选中一级标题**（如"1 引言"、"2 总体设计"等）
2. **设置字体**：
   - 字体：黑体
   - 字号：三号（16磅）
   - 加粗
   
3. **设置段落格式**：
   - 行距：固定值 29磅
   - 无缩进
   
### 3. 其他标题设置

**步骤：**

1. **选中二级、三级标题**（如"1.1 目的"、"1.1.1 范围"等）
2. **设置字体**：
   - 字体：仿宋GB2312
   - 字号：三号（16磅）
   
3. **设置段落格式**：
   - 行距：固定值 29磅
   - 无缩进

## 二、三线格表格设置

### 三线格表格定义

三线格是学术论文、技术文档的标准表格格式：
- 顶线：粗线（1.5磅）
- 底线：粗线（1.5磅）
- 表头下方：细线（1磅）
- 其他边框：无

### 详细操作步骤

**步骤1：创建表格**

1. 插入 → 表格 → 选择行列数
2. 填写表格内容

**步骤2：设置三线格边框**

1. **选中整个表格**
2. 右键 → 表格属性 → 边框和底纹
3. **设置边框**：
   - 点击"无"清除所有边框
   - 点击"上边框"，设置：1.5磅，黑色
   - 点击"下边框"，设置：1.5磅，黑色
4. **确定**

**步骤3：设置表头行边框**

1. **选中表头行**（第一行）
2. 右键 → 边框和底纹
3. **只保留下边框**：
   - 点击"无"清除所有边框
   - 点击"下边框"，设置：1磅，黑色
4. **确定**

**步骤4：设置表头字体**

1. **选中表头单元格**
2. 设置：
   - 字体：黑体
   - 字号：四号（14磅）
   - 加粗
   - 对齐：居中

**步骤5：设置正文单元格**

1. **选中正文单元格**
2. 设置：
   - 字体：宋体
   - 字号：四号（14磅）
   - 对齐：左对齐或居中（根据内容）

### 批量应用表格样式

**创建表格样式模板：**

1. 设计好一个三线格表格
2. 选中表格 → 表格工具 → 设计 → 表格样式
3. 点击"新建表格样式"
4. 命名为"三线格样式"
5. 保存
6. 以后选中其他表格 → 应用"三线格样式"

## 三、图片处理方法

### 1. 线框图重新绘制

**工具推荐：**
- **Visio**：专业流程图工具（推荐）
- **Draw.io**：免费在线工具（https://app.diagrams.net）
- **PowerPoint**：简单图表绘制
- **Lucidchart**：在线协作工具

**绘制原则：**
- 美观为第一原则
- 使用统一的风格
- 清晰的标注
- 无乱码字体

### 2. 使用Visio重新绘制

**步骤：**

1. **打开Visio**
2. **新建流程图/线框图**
3. **绘制图表**：
   - 使用标准形状
   - 统一字体（建议宋体、黑体）
   - 清晰的连接线
   - 添加编号和说明
4. **导出为PNG**：
   - 文件 → 导出 → PNG
   - 设置分辨率：300 DPI（高清）
5. **插入Word文档**：
   - 插入 → 图片 → 选择PNG文件
   - 调整大小和位置

### 3. 使用Draw.io绘制

**步骤：**

1. 访问 https://app.diagrams.net
2. 新建图表
3. 绘制流程图
4. 导出为PNG（高清）
5. 插入Word

## 四、表述修正为将来时态

### 修正原则

**所有内容统一为将来时态**：即所有内容都是未来要做的，还没实现。

### 修正规则

| 原表述 | 修正为 | 说明 |
|--------|--------|------|
| 已完成 | 将完成 | 表示未来将完成 |
| 已实现 | 将实现 | 表示未来将实现 |
| 正在进行 | 将进行 | 表示未来将进行 |
| 已经 | 将要 | 表示未来将要 |
| 正在 | 将要 | 表示未来将要 |
| 实现了 | 将实现 | 表示未来将实现 |
| 完成了 | 将完成 | 表示未来将完成 |
| 开发了 | 将开发 | 表示未来将开发 |
| 设计了 | 将设计 | 表示未来将设计 |
| 测试了 | 将测试 | 表示未来将测试 |
| 部署了 | 将部署 | 表示未来将部署 |

### 批量修正方法

**方法1：查找替换**

1. Ctrl+H 打开查找替换
2. 输入原表述和新表述
3. 全部替换

**方法2：使用Python脚本**

```python
# 见 format_word.py 脚本
python format_word.py input.docx output.docx
```

## 五、目录生成方法

### 自动生成目录

**步骤：**

1. **确认标题样式**：
   - 所有标题必须使用Word内置样式（标题1、标题2等）
   - 或自定义样式并设置大纲级别
   
2. **插入目录**：
   - 光标定位到目录页
   - 引用 → 目录 → 自动目录
   - 或选择"自定义目录"
   
3. **设置目录格式**：
   - 右键目录 → 编辑域
   - 选项 → 目录 → 格式：来自模板
   - 或使用"正式"格式
   
4. **更新目录**：
   - 右键目录 → 更新域
   - 选择"更新整个目录"

### 目录格式要求

- 目录标题：黑体，三号
- 一级目录：黑体，小三号
- 二级目录：宋体，四号
- 三级目录：宋体，小四号
- 行距：固定值 20磅

## 六、页面设置

### 标准页面格式

**步骤：**

1. **布局 → 页面设置**
2. **纸张**：
   - 纸张大小：A4
   - 方向：纵向
   
3. **页边距**：
   - 上：2.5厘米
   - 下：2.5厘米
   - 左：3厘米
   - 右：2.5厘米
   
4. **版式**：
   - 页眉：1.5厘米
   - 页脚：1.75厘米

---

## 七、页眉页脚设置

### 页眉设置

**步骤：**

1. **插入页眉**：
   - 插入 → 页眉 → 编辑页眉
   - 输入："软件设计说明书"
   
2. **格式设置**：
   - 字体：宋体，小五号
   - 对齐：居中
   
3. **设置页眉线**：
   - 选中页眉文字
   - 开始 → 边框 → 下边框

### 页脚设置

**步骤：**

1. **插入页码**：
   - 插入 → 页码 → 页面底端
   - 选择"普通数字2"
   
2. **格式设置**：
   - 字体：宋体，小五号
   - 对齐：居中
   
3. **页码格式**：
   - 格式 → 页码格式
   - 编号格式：1, 2, 3...
   - 起始页码：1

## 八、完整操作流程

### 推荐操作顺序

```
步骤1: 备份原文档
    └─ 复制文档，命名为"备份_原文档.docx"

步骤2: 设置页面格式
    └─ 页边距、纸张大小、页眉页脚

步骤3: 处理字体格式

    ├─ 设置其他标题格式

步骤4: 处理表格格式
    ├─ 选中每个表格

步骤5: 处理图片

    ├─ 格式检查：字体、行距、缩进
    ├─ 表格检查：三线格、字体
    ├─ 图片检查：清晰度、美观度
    ├─ 表述检查：将来时态
    ├─ 目录检查：更新、格式
```

## 十、常见问题解决

### 问题1：字体找不到"仿宋GB2312"

**解决方法**：
- 使用"仿宋"代替
- 或下载并安装"仿宋GB2312"字体

### 问题2：三线格表格设置后显示不对

**解决方法**：
- 清除所有边框后重新设置
- 确保选中正确的行或单元格
- 检查边框磅数是否正确

### 问题3：目录更新后格式错乱

**解决方法**：
- 删除目录重新插入
- 或手动调整目录格式
- 确保标题样式正确

### 问题4：图片太大或太小

**解决方法**：
- 右键图片 → 大小和位置
- 锁定纵横比
- 调整到合适大小（建议宽度15厘米）
