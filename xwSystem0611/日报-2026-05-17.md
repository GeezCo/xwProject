## 二、技术要点

### Docker Compose V2 语法
- 服务器使用 Docker Compose V2 插件
- 命令格式：`docker compose`（非 `docker-compose`）
- 已更新 `setup.sh` 中所有相关命令

### 离线镜像制作流程
```bash
# 1. 提交运行中的容器为镜像
docker commit xianwei-algorithm xianwei-algorithm:offline

# 2. 导出镜像为 tar 文件
docker save -o xianwei-algorithm-offline.tar xianwei-algorithm:offline

# 3. 在离线机器上加载镜像
docker load -i xianwei-algorithm-offline.tar
```
