# 删除接口文档

## 新增接口列表

### 1. 删除单个分类
**接口**: `DELETE /uygur/category/{categoryId}`

**参数**:
- `categoryId` (路径参数): 分类ID

**返回**:
```json
{
  "code": 1,
  "data": "删除成功",
  "msg": "success",
  "flag": "true"
}
```

**示例**:
```bash
curl -X DELETE http://localhost:8081/uygur/category/59
```

---

### 2. 批量删除分类
**接口**: `POST /uygur/category/batchDelete`

**参数**:
- Body (JSON数组): 分类ID列表

**返回**:
```json
{
  "code": 1,
  "data": "成功删除2个分类",
  "msg": "success",
  "flag": "true"
}
```

**示例**:
```bash
curl -X POST http://localhost:8081/uygur/category/batchDelete \
  -H "Content-Type: application/json" \
  -d '[61, 65]'
```

---

### 3. 删除单个报文
**接口**: `DELETE /uygur/text/{sid}`

**参数**:
- `sid` (路径参数): 报文ID

**返回**:
```json
{
  "code": 1,
  "data": "删除成功",
  "msg": "success",
  "flag": "true"
}
```

**示例**:
```bash
curl -X DELETE http://localhost:8081/uygur/text/188
```

---

### 4. 批量删除报文
**接口**: `POST /uygur/text/batchDelete`

**参数**:
- Body (JSON数组): 报文ID列表

**返回**:
```json
{
  "code": 1,
  "data": "成功删除5条报文",
  "msg": "success",
  "flag": "true"
}
```

**示例**:
```bash
curl -X POST http://localhost:8081/uygur/text/batchDelete \
  -H "Content-Type: application/json" \
  -d '[188, 189, 190]'
```

---

### 5. 删除分类及其下所有报文
**接口**: `DELETE /uygur/category/{categoryId}/withTexts`

**参数**:
- `categoryId` (路径参数): 分类ID

**返回**:
```json
{
  "code": 1,
  "data": "成功删除分类及其下8121条报文",
  "msg": "success",
  "flag": "true"
}
```

**说明**: 
- 先删除该分类下的所有报文
- 再删除分类本身
- 返回删除的报文数量

**示例**:
```bash
curl -X DELETE http://localhost:8081/uygur/category/57/withTexts
```

---

## 修改的文件

### 后端文件
1. **UygurMapper.java** - 新增5个删除方法
   - `deleteCategory(Integer categoryId)` - 删除单个分类
   - `deleteCategoriesBatch(List<Integer> categoryIds)` - 批量删除分类
   - `deleteText(Long sid)` - 删除单个报文
   - `deleteTextsBatch(List<Long> sids)` - 批量删除报文
   - `deleteTextsByType(Integer typeId)` - 删除分类下所有报文

2. **UygurService.java** - 新增5个服务接口方法

3. **UygurServiceImpl.java** - 实现5个服务方法
   - 所有删除方法都添加了 `@Transactional` 注解保证事务一致性

4. **UygurController.java** - 新增5个REST接口
   - `DELETE /uygur/category/{categoryId}`
   - `POST /uygur/category/batchDelete`
   - `DELETE /uygur/text/{sid}`
   - `POST /uygur/text/batchDelete`
   - `DELETE /uygur/category/{categoryId}/withTexts`
