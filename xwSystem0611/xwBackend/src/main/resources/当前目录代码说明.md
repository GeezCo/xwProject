# resources

Spring Boot 配置文件和 MyBatis 映射文件。

## 文件说明

| 文件 | 说明 |
|------|------|
| application.yml | Spring Boot 主配置：数据源（MySQL 8010）、Druid 连接池、MyBatis、日志、超时设置 |

## 子目录

| 目录 | 说明 |
|------|------|
| mapper/ | MyBatis XML 映射文件（FusionMapper.xml） |
| db/ | 数据库 DDL 脚本 |

## db/ 脚本说明

| 文件 | 说明 |
|------|------|
| extraction_ddl.sql | extraction_result 表建表语句 |
| fusion_report_ddl.sql | fusion_report 表建表语句 |
| origin_text_modal_type_ddl.sql | origin_text 表添加 modal_type 字段 |
