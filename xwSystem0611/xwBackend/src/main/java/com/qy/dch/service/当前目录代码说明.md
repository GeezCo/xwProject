# service

业务逻辑层，定义接口并在 impl/ 子目录中实现。

## 文件说明

| 文件 | 说明 |
|------|------|
| UygurService.java | 报文管理服务接口：分类查询、报文列表、详情、导入、重置 |
| ExtractionService.java | 属性抽取服务接口：执行抽取、查询结果 |
| FusionService.java | 报文融合服务接口：创建融合、保存、列表、详情 |

## 子目录

| 目录 | 说明 |
|------|------|
| impl/ | Service 接口实现类 |

## impl/ 文件说明

| 文件 | 说明 |
|------|------|
| UygurServiceImpl.java | 报文管理实现，调用 UygurMapper 进行数据库操作 |
| ExtractionServiceImpl.java | 属性抽取实现，通过 HTTP 调用算法服务 /extract 接口，解析结果后存入数据库 |
| FusionServiceImpl.java | 报文融合实现，通过 HTTP 调用算法服务 /fusion/create 接口，组装融合报告 |
