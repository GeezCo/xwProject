# mapper

MyBatis 数据访问层，定义 SQL 映射接口。

## 文件说明

| 文件 | 说明 |
|------|------|
| UygurMapper.java | 报文数据访问：查询分类目录、分页查询报文列表、查询报文详情、插入文本、重置抽取状态 |
| ExtractionMapper.java | 抽取结果数据访问：按 originTextId 查询、插入/更新抽取结果 |
| FusionMapper.java | 融合报告数据访问：插入、更新、按 ID 查询、分页列表查询 |

## 对应 XML 映射

MyBatis XML 映射文件位于 `src/main/resources/mapper/`：
- FusionMapper.xml（其余使用注解方式定义 SQL）
