# request

前端请求参数对象，用于接收 Controller 入参。

## 文件说明

| 文件 | 说明 |
|------|------|
| GetListRequest.java | 报文列表查询参数：pageNum, pageSize, typeId, modalType |
| FusionCreateRequest.java | 融合创建请求：reports 列表、fusionType 融合类型、customTitle 自定义标题 |
| ReportData.java | 融合时的单篇报文数据结构：id, title, content, times, type, extractionResult |
| AddCategoryRequest.java | 添加分类请求：categoryName, parentCategoryName |
