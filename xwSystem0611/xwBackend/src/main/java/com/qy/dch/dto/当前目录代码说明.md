# dto

数据传输对象，用于 Controller 和 Service 之间传递数据。

## 文件说明

| 文件 | 说明 |
|------|------|
| OriginTextDTO.java | 原始报文：sid, title, content, times, type, modalType 等字段 |
| ExtractionResultDTO.java | 抽取结果：originTextId, eventsJson, labelsJson, entitiesJson, totalEvents, status |
| FusionDTO.java | 融合报告：fusionId, title, summary, timeline, content, entities, labels, sourceIds |
| TextTypeDTO.java | 分类目录：id, typeName |
| PageResultDTO.java | 分页结果：total, list, pageNum, pageSize |
| AddCategoryResultDTO.java | 添加分类结果：categoryId, categoryName, parentId, parentName, isNewParent |
| ImportResultDTO.java | 批量导入结果：totalLines, successCount, failCount, categoryId, errors |
