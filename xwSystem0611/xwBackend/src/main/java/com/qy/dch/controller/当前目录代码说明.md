# controller

REST API 控制器层，接收前端 HTTP 请求并调用 Service 层处理。

## 文件说明

| 文件 | 说明 |
|------|------|
| UygurController.java | 报文管理接口：分类目录查询、报文列表分页查询、报文详情、批量导入、重置抽取状态 |
| ExtractionController.java | 属性抽取接口：执行 LLM 事件抽取（POST /extraction/extract）、查询抽取结果（GET /extraction/result/{id}） |
| FusionController.java | 报文融合接口：创建融合报告、保存、列表查询、详情查询 |

## 接口汇总

| 路径 | 方法 | Controller | 功能 |
|------|------|------------|------|
| /uygur/category | GET | Uygur | 分类目录列表 |
| /uygur/getTextList | POST | Uygur | 分页查询报文 |
| /uygur/detail/{sid} | GET | Uygur | 报文详情 |
| /extraction/extract | POST | Extraction | 执行事件抽取 |
| /extraction/result/{id} | GET | Extraction | 查询抽取结果 |
| /api/fusion/create | POST | Fusion | 创建融合报告 |
| /api/fusion/save | POST | Fusion | 保存融合报告 |
| /api/fusion/list | GET | Fusion | 融合报告列表 |
| /api/fusion/detail/{id} | GET | Fusion | 融合报告详情 |
