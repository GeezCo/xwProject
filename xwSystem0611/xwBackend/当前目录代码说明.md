# 后端

Spring Boot 2.7 后端服务，提供报文管理、属性抽取、报文融合的 REST API。

## 文件说明

| 文件 | 说明 |
|------|------|
| pom.xml | Maven 项目配置，依赖 Spring Boot、MyBatis、Druid、MySQL、MinIO |
| mvnw / mvnw.cmd | Maven Wrapper 脚本，无需系统安装 Maven 即可构建 |

## 子目录

| 目录 | 说明 |
|------|------|
| src/main/java/com/qy/dch/ | Java 源码根目录 |
| src/main/resources/ | 配置文件、MyBatis XML 映射、DDL 脚本 |
| src/test/ | 单元测试 |
| logs/ | 运行时日志输出 |

## 启动方式

```bash
# 开发环境
./mvnw spring-boot:run

# 生产环境（配合 deploy/.env 环境变量）
java -jar uygur-project-0.0.1-SNAPSHOT.jar --spring.config.location=application-prod.yml
```

## 端口

- 开发环境：8081（application.yml）
- 生产环境：9202（通过 .env 的 BACKEND_PORT 配置）

## 数据库

MySQL 8.0，数据库 `uygur_project`，4 张表：`origin_text`、`text_type`、`extraction_result`、`fusion_report`
