# 五、算法服务接口 (Flask app.py)

> 以下接口由后端内部调用，前端不直接访问。

## 5.1 健康检查

```
GET http://localhost:5001/health
```

**响应**:
```json
{
  "status": "ok",
  "service": "algorithm-service",
  "time": "2026-04-22 10:00:00"
}
```

## 5.4 报文融合

```
POST http://localhost:5001/fusion/create
Content-Type: application/json
```

**请求体**:
```json
{
  "reports": [
    {
      "id": 9741,
      "title": "标题",
      "content": "正文",
      "extractionResult": {
        "events": [...],
        "entities": {...},
        "labels": [...]
      }
    }
  ],
  "fusionType": "standard",
  "customTitle": null
}
```

**响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "fusionId": 1713766800000,
    "title": "融合报告标题",
    "summary": "摘要",
    "timeline": [
      {
        "time": "2024-03-15",
        "description": "美军第七舰队在菲律宾海派遣里根号航母执行巡航任务"
      }
    ],
    "content": "## 详细内容(Markdown)",
    "entities": [
      {
        "category": "人物",
        "name": "美国总统拜登",
        "action": "2024-03-15: 拜登在白宫签署关于加强印太地区军事部署的行政命令"
      },
      {
        "category": "船只",
        "name": "里根号航母",
        "action": "2024-03-15: 美军派遣里根号航母前往菲律宾海执行巡航任务"
      }
    ],
    "labels": ["军事", "国际"],
    "sourceIds": [9741, 9742],
    "modelUsed": "Qwen3.5-122B-A10B",
    "createTime": "2026-04-22 10:30:00",
    "updateTime": "2026-04-22 10:30:00"
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| timeline[].time | string | 标准化日期 `YYYY-MM-DD` |
| timeline[].description | string | **事件原文**（取自 event.original_text，缺失时回退 action）；同日多事件用 `; ` 拼接，最多3条 |
| entities | array | **目标列表表格行数组**（替代旧版字典格式） |
| entities[].category | string | 实体类别（船只/飞机/人物等11类） |
| entities[].name | string | 实体名称（同义实体已合并，展示最长形式） |
| entities[].action | string | **`时间: 原文` 拼接**，多事件用顿号分隔；无关联事件时为 `-` |
| labels | array | 智能标签（去重并截前8个） |
| sourceIds | array | 源报文ID数组 |

**关键变更**（2026-06-02）:

1. **entities 结构重构**：从分类字典改为表格行数组
   - 旧：`{"人物": ["拜登"], "组织": ["北约"]}`
   - 新：`[{category: "人物", name: "拜登", action: "..."}]`

2. **聚合内容改为 original_text**：
   - `timeline[].description` 取自 `event.original_text`（带兜底 action）
   - `entities[].action` 也取自 `event.original_text`

3. **实体智能归一化**：
   - 标点/全半角差异：`F-35` 与 `F－35` 合并
   - 职位前缀：`拜登` 与 `美国总统拜登` 合并（展示最长形式）
   - 装备后缀：`里根号` 与 `里根号航母` 合并
   - 括号补充：`F-35(闪电Ⅱ)` 与 `F-35` 合并
   - 词典：`_ENTITY_PREFIXES` / `_ENTITY_SUFFIXES`（fusion_extractor.py 顶部）

4. **action 时间排序**：同一实体多事件按时间升序拼接，时间为空的排最后

## 5.5 事件拆分与分析

```
POST http://localhost:5001/eventSplit
Content-Type: application/json
```

**请求体**:
```json
{
  "text": "报文正文内容..."
}
```

**响应**:
```json
{
  "events": [
    {
      "time": "2024年3月15日",
      "location": "美国白宫",
      "event_content": "美国总统拜登签署关于人工智能的行政命令",
      "event_analysis": "该事件发生于2024年3月15日，地点位于美国政治核心区域白宫..."
    }
  ],
  "overall_analysis": "整体综合分析文本..."
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| events | array | 识别出的独立事件列表 |
| events[].time | string | 事件时间（LLM提取） |
| events[].location | string | 事件地点 |
| events[].event_content | string | 事件核心内容（一句话摘要） |
| events[].event_analysis | string | 单事件深度分析（涵盖时间、空间、主体、行为、影响、关联维度） |
| overall_analysis | string | 整体综合分析（趋势判断、风险预警、战略建议） |

**说明**:
- 文本长度限制：最大10000字
- 自动重试机制：最多3次
- LLM模型：Qwen3.5-35B-A3B
- 响应时间：约10秒/篇
- 由后端定时任务 `DailyAnalysisTask` 调用，前端不直接访问
