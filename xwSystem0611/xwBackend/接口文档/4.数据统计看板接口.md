# 四、数据统计看板接口 (DashboardController) ✨ 新增

## 4.1 核心指标概览

```
GET /api/dashboard/overview
```

**响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "totalReports": 6061,
    "extractedReports": 1234,
    "fusionReports": 45,
    "totalLabels": 328
  }
}
```

**字段说明**:
- `totalReports`: 总报文数（origin_text 表总数）
- `extractedReports`: 已抽取报文数（is_extracted=1）
- `fusionReports`: 融合报告数（fusion_report 表总数）
- `totalLabels`: 标签总数（extraction_result 表中 labels_json 去重统计）

## 4.2 报文分类分布

```
GET /api/dashboard/categoryDistribution
```

**响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": [
    { "type": "港口", "cnt": 523 },
    { "type": "机场", "cnt": 412 },
    { "type": "舰机过航", "cnt": 1205 },
    { "type": "军事演习", "cnt": 876 }
  ]
}
```

**说明**:
- SQL: `SELECT type, COUNT(*) as cnt FROM origin_text GROUP BY type`
- 返回各二级分类下的报文数量

## 4.3 报文模态分布

```
GET /api/dashboard/modalDistribution
```

**响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": [
    { "modal_type": "文字报", "cnt": 4523 },
    { "modal_type": "图文报", "cnt": 1438 },
    { "modal_type": "声像报", "cnt": 100 }
  ]
}
```

**说明**:
- SQL: `SELECT modal_type, COUNT(*) as cnt FROM origin_text GROUP BY modal_type`
- 返回各模态类型的报文数量

## 4.4 最近融合报告列表

```
GET /api/dashboard/recentFusions
```

**响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": [
    {
      "fusionId": 123,
      "title": "多国军事演习综合分析报告",
      "summary": "本报告综合分析...",
      "createTime": "2026-05-18 14:30:00"
    },
    {
      "fusionId": 122,
      "title": "第七舰队相关报文融合",
      "summary": "...",
      "createTime": "2026-05-17 10:15:00"
    }
  ]
}
```

**说明**:
- 返回最近若干条融合报告（按当前实现取列表前N条）
- 字段：`fusionId`、`title`、`summary`、`createTime`
- 按创建时间倒序排列
- ⚠️ 当前实现 **不返回** `sourceCount` 字段，如需源报文数请通过 3.4 详情接口查询 `sourceIds` 后计算
