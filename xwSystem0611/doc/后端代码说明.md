# 后端代码说明

献微系统后端采用 Spring Boot 2.7.18 + MyBatis 2.3.2 + Druid 1.2.20 + MySQL 8.0 技术栈，提供报文管理、属性抽取、报文融合和数据统计等核心服务。

---

## 一、项目概览

### 技术栈
- **核心框架**: Spring Boot 2.7.18
- **持久层**: MyBatis 2.3.2
- **数据库**: MySQL 8.0.33
- **连接池**: Druid 1.2.20
- **JSON处理**: Fastjson 1.2.83
- **日志**: Logback (Spring Boot 默认)

## 二、分层架构

### 2.1 Controller 层
**职责**：接收 HTTP 请求，参数校验，调用 Service 层，返回统一响应

**规范**：
- 使用 `@RestController` 注解
- 使用 `@RequestMapping` 定义路由前缀
- 返回类型统一为 `ResultVO`
- 使用 `@Slf4j` 记录日志

### 2.2 Service 层
**职责**：业务逻辑处理，事务管理，调用 Mapper 层

**规范**：
- 接口与实现分离（`UygurService` 接口 + `UygurServiceImpl` 实现类）
- 使用 `@Service` 注解
- 使用 `@Transactional` 管理事务
- 复杂业务逻辑封装在 Service 层

### 2.3 Mapper 层
**职责**：数据库访问，执行 SQL 语句

**规范**：
- 使用 `@Mapper` 注解
- 使用 MyBatis 注解式 SQL（`@Select`, `@Insert`, `@Update`, `@Delete`）
- 动态 SQL 使用 `<script>` 标签

### 2.4 DTO/Request/Domain 层
**职责**：数据传输对象，解耦前后端数据结构

```java
@PostMapping("/getTextList")
public ResultVO getList(@RequestBody GetListRequest getListRequest) {
    PageResultDTO<OriginTextDTO> pageResult = uygurService.getTextListPaged(getListRequest);
    
    PageDomain pageDomain = new PageDomain();
    pageDomain.setPages(pageResult.getTotalPages());
    pageDomain.setTotal((int) pageResult.getTotal());
    
    // 转换 DTO 为前端格式，解析 labelsJson 为数组
    List<Map<String, Object>> shows = new ArrayList<>();
    for (OriginTextDTO dto : pageResult.getList()) {
        Map<String, Object> item = new HashMap<>();
        item.put("sid", dto.getSid());
        item.put("title", dto.getTitle());
        // ... 其他字段
        if (dto.getLabelsJson() != null && !dto.getLabelsJson().isEmpty()) {
            JSONArray labelsArray = JSON.parseArray(dto.getLabelsJson());
            item.put("labels", labelsArray);
        } else {
            item.put("labels", new JSONArray());
        }
        shows.add(item);
    }
    pageDomain.setList(shows);
    
    return ResultVO.success(pageDomain);
}
```

```java
@PostMapping("/importFromJsonl")
public ResultVO importFromJsonl(
        @RequestParam("file") MultipartFile file,
        @RequestParam("parentCategoryName") String parentCategoryName,
        @RequestParam("categoryName") String categoryName) {
    
    // 参数校验
    if (file.isEmpty()) {
        return ResultVO.error("文件不能为空");
    }
    
    // 调用 Service 层执行导入
    ImportResultDTO result = uygurService.importFromJsonl(file, parentCategoryName, categoryName);
    
    if (!result.getErrors().isEmpty() && result.getSuccessCount() == 0) {
        return ResultVO.error("导入失败: " + result.getErrors().get(0));
    }
    
    return ResultVO.success(result);
}
```

```java
@PostMapping("/extract")
public ResultVO extract(@RequestBody Map<String, Object> params) {
    Integer originTextId = (Integer) params.get("originTextId");
    Boolean force = (Boolean) params.getOrDefault("force", false);
    
    // 检查是否已抽取
    Integer isExtracted = uygurMapper.selectIsExtracted(originTextId);
    if (isExtracted != null && isExtracted == 1 && !force) {
        return ResultVO.error("该报文已抽取，如需重新抽取请设置 force=true");
    }
    
    // 获取报文内容
    OriginTextDTO text = uygurMapper.selectById(Long.valueOf(originTextId));
    
    // 调用算法服务（HTTP 请求 Flask）
    ExtractionResultDTO result = extractionService.extractEvents(text.getContent(), originTextId);
    
    // 保存抽取结果
    extractionService.saveExtractionResult(originTextId, result);
    
    // 更新抽取状态
    uygurMapper.updateExtractedStatus(originTextId, 1);
    
    return ResultVO.success(result);
}
```

```java
@Override
public List<Map<String, Object>> getCategoryDistribution() {
    // 原始统计数据（type为ID）
    List<Map<String, Object>> distribution = uygurMapper.countByType();
    
    // 查询所有分类构建ID->名称映射
    List<TextTypeDTO> allCategories = uygurMapper.getCategories();

    for (TextTypeDTO cat : allCategories) {
        idToNameMap.put(cat.getId(), cat.getTypeName());
    }
    
    // 替换type字段为typeName
    for (Map<String, Object> item : distribution) {

        if (typeObj instanceof Number) {
            Integer typeId = ((Number) typeObj).intValue();
            String typeName = idToNameMap.getOrDefault(typeId, "未分类");
            item.put("type", typeName);
        }
    }
    
    return distribution;
```

#### 4.1.1 JSONL 导入
```java
@Override
@Transactional
public ImportResultDTO importFromJsonl(MultipartFile file, String parentCategoryName, String categoryName) {
    ImportResultDTO result = new ImportResultDTO();
    
    // 1. 查询分类ID
    TextTypeDTO parentCategory = uygurMapper.getCategoryByNameAndParent(parentCategoryName, null);
    TextTypeDTO category = uygurMapper.getCategoryByNameAndParent(categoryName, parentCategory.getId());
    Integer categoryId = category.getId();
    
    // 2. 读取JSONL文件
    List<OriginTextDTO> textList = new ArrayList<>();
    try (BufferedReader reader = new BufferedReader(
            new InputStreamReader(file.getInputStream(), StandardCharsets.UTF_8))) {
        
        String line;
        int lineNum = 0;
        while ((line = reader.readLine()) != null) {
            lineNum++;
            if (line.trim().isEmpty()) continue;
            
            try {
                JSONObject json = JSON.parseObject(line);
                String title = json.getString("标题");
                String content = json.getString("内容");
                String timeStr = json.getString("时间");
                
                // 转换时间格式
                String mysqlTime = convertTimeFormat(timeStr);
                
                // 构造DTO
                OriginTextDTO dto = new OriginTextDTO();
                dto.setTitle(title);
                dto.setContent(content);
                dto.setTimes(mysqlTime);
                dto.setType(categoryId);
                dto.setModalType("文字报");
                
                textList.add(dto);
                result.setSuccessCount(result.getSuccessCount() + 1);
                
                // 每100条批量插入一次
                if (textList.size() >= 100) {
                    uygurMapper.batchInsertTexts(textList);
                    textList.clear();
                }
            } catch (Exception e) {
                result.getErrors().add("第" + lineNum + "行解析失败: " + e.getMessage());
                result.setFailCount(result.getFailCount() + 1);
            }
        }
        
        // 插入剩余数据
        if (!textList.isEmpty()) {
            uygurMapper.batchInsertTexts(textList);
        }
    } catch (IOException e) {
        result.getErrors().add("文件读取失败: " + e.getMessage());
    }
    
    return result;
}
```

#### 4.1.2 时间格式转换
```java
private String convertTimeFormat(String timeStr) {
    if (timeStr.contains("年")) {
        // "2026年01月01日" -> "2026-01-01"
        DateTimeFormatter inputFormatter = DateTimeFormatter.ofPattern("yyyy年MM月dd日");
        DateTimeFormatter outputFormatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
        LocalDate date = LocalDate.parse(timeStr, inputFormatter);
        return date.format(outputFormatter);
    }
    return timeStr;
}
```

```java
public ExtractionResultDTO extractEvents(String content, Integer originTextId) {
    String algorithmUrl = algorithmBaseUrl + "/extract";
    
    // 构造请求体
    Map<String, Object> body = new HashMap<>();
    body.put("text", content);
    body.put("origin_text_id", originTextId);
    
    // HTTP 调用 Flask 算法服务（超时设置 20 分钟）
    HttpResponse<String> response = HttpClient.newHttpClient()
        .send(HttpRequest.newBuilder()
            .uri(URI.create(algorithmUrl))
            .timeout(Duration.ofMinutes(20))
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(JSON.toJSONString(body)))
            .build(), HttpResponse.BodyHandlers.ofString());
    
    // 解析响应
    return JSON.parseObject(response.body(), ExtractionResultDTO.class);
}
```

```java
        "<if test='typeIds != null and typeIds.size() > 0'>" +
        "AND o.type IN " +
        "<foreach collection='typeIds' item='id' open='(' separator=',' close=')'>" +
```

## 十、调用链路

### 10.1 属性抽取流程

```
前端 searchResult.vue
  ↓ POST /extraction/extract
后端 ExtractionController.extract()
  ↓ extractionService.extractEvents()
后端 ExtractionServiceImpl
  ↓ HTTP POST http://localhost:5001/extract
算法 Flask /extract
  ↓ LLMEventExtractorV3.extract()
算法 调用 LLM API
  ↓ 返回事件JSON
算法 调用 RexUniNlu 分类
  ↓ 返回标签
算法 返回完整结果
后端 保存到 extraction_result 表
后端 更新 origin_text.is_extracted=1
前端 显示抽取结果
```

### 10.2 报文融合流程

```
前端 fusionResult.vue
  ↓ POST /api/fusion/create (含源报文ID列表)
后端 FusionController.createFusion()
  ↓ 收集源报文+抽取结果
后端 HTTP POST http://localhost:5001/fusion/create
算法 Flask /fusion/create
  ↓ FusionExtractor.create_fusion()
算法 数据预处理 → 信息整合 → LLM生成 → 结果组装
算法 返回融合报告 JSON
后端 返回前端展示
```

### 10.3 JSONL 导入流程

```
前端 Dashboard.vue
  ↓ POST /uygur/importFromJsonl (multipart/form-data)
后端 UygurController.importFromJsonl()
  ↓ uygurService.importFromJsonl()
后端 UygurServiceImpl
  ↓ 1. 校验分类（一级、二级）
  ↓ 2. 逐行读取JSONL
  ↓ 3. 解析"标题"/"内容"/"时间"字段
  ↓ 4. 时间格式转换
  ↓ 5. 每100条批量插入 origin_text
后端 返回导入结果
前端 显示成功/失败统计
```

## 十二、注意事项

1. **长超时**：调用算法服务的 HTTP 请求设置 20 分钟超时（适配 LLM 处理）
2. **批量插入**：JSONL 导入采用每 100 条批量插入策略，提升导入性能
3. **事务管理**：涉及多表操作的方法使用 `@Transactional` 注解
4. **异常处理**：Service 层抛出业务异常，Controller 层捕获并返回友好错误信息
5. **CORS 配置**：开发环境允许所有域名，生产环境建议限制特定域名
6. **MinIO 配置**：图片访问地址通过 `/uygur/config` 接口动态返回，避免前端硬编码
7. **分类层级**：text_type 表通过 `parent_id` 字段维护父子关系，NULL 表示一级分类
8. **批量任务**：批量抽取任务使用 `ConcurrentHashMap` 管理状态，支持取消和进度查询
