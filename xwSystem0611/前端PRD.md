# 献微系统 前端产品需求文档

## 项目名称

献微系统 (Xianwei System)

---

## 主要功能

- 报文管理
  - 文本搜索
  - 分类筛选
  - 时间排序
  - 详情查看
  - 标签编辑

- 属性抽取
  - 一键抽取
  - 进度提示
  - 结果展示
  - 重新抽取
  - 结果存储

- 事件拆分
  - 多事件拆分
  - 要素补全
  - 指代消解
  - 结果导出

- 数据生成
  - 样例导入
  - 批量生成
  - 多样性控制

---

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue.js | 2.5.x | 前端框架 |
| Element UI | 2.15.x | UI组件库 |
| Vue Router | 3.x | 路由管理 |
| Axios | 0.27.x | HTTP请求 |
| ECharts | 5.x | 数据可视化 |

---

## 项目特性

- **前后端分离**：Vue前端 + Spring Boot后端
- **Token鉴权**：localStorage存储 + 路由守卫
- **响应式布局**：适配不同屏幕尺寸
- **状态反馈**：加载动画 + 操作提示
- **错误处理**：友好提示 + 自动重试
- **性能优化**：虚拟滚动 + 懒加载
