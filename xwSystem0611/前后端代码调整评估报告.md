**关键代码** (Line 198-225):
```javascript
const entitiesTableData = computed(() => {
  const ent = entitiesData.value
  if (!ent) return []

  // 新数据结构：直接是表格行数组 [{category, name, action}, ...]
  if (Array.isArray(ent)) {
    return ent.map(row => ({
      category: row.category || '',
      name: row.name || '',
      action: row.action || '-'
    }))
  }

  // 向后兼容：老数据结构是字典 {category: [names]}
  const rows = []
  for (const [category, names] of Object.entries(ent)) {
    if (names && names.length) {
      names.forEach(name => {
        rows.push({
          category,
          name,
          action: '-'
        })
      })
    }
  }
  return rows
})
```

**分析**:
- ✅ 使用 **动态对象遍历** (`Object.entries`)
- ✅ 支持新格式（数组）和旧格式（字典）的自动兼容
- ✅ 分类名称来自数据本身，不依赖硬编码
- ✅ 目标列表表格的 `category` 列动态显示分类名称
