## 布局说明

### 整体结构
- **单页滚动布局**: 所有统计卡片垂直排列，支持页面滚动
- **响应式网格**: 使用 CSS Grid 实现自适应布局
- **卡片化设计**: 每个统计模块独立成卡片，带阴影和圆角

### 区域划分

#### 1. 页面标题区
- 左侧: 📊 图标 + "数据统计看板" 标题
- 右侧: 最后更新时间（实时刷新）
- 高度: 60px
- 背景: `var(--bg-panel)`

#### 2. 核心指标卡片区
- 布局: 4卡片横排（Grid 4列）
- 每个卡片包含:
  - 图标 + 指标名称
  - 大号数字（主指标）
  - 对比数据（较昨日变化，绿色↑/红色↓）
- 卡片尺寸: 等宽，高度 120px
- 间距: 16px

#### 3. 图表区域（2x3网格）
- **第一行**: 报文分类分布（饼图） + 报文模态分布（环形图）
- **第二行**: 报文时间分布（折线图+柱状图，跨2列）
- **第三行**: 抽取进度统计 + 高频标签云
- **第四行**: 实体类型分布（横向柱状图，跨2列）
- **第五行**: 融合报告统计（表格，跨2列）

## 交互说明

### 页面加载
1. 进入页面自动加载所有统计数据
2. 显示 loading 状态（骨架屏或加载动画）
3. 数据加载完成后渐入显示

### 数据刷新
1. 页面右上角显示"最后更新时间"
2. 提供"刷新"按钮，点击重新加载所有数据
3. 可选: 自动刷新（每5分钟）

### 图表交互
1. **饼图/环形图**: 鼠标悬停显示具体数值和百分比
2. **折线图**: 鼠标悬停显示当日数据
3. **柱状图**: 点击柱子可跳转到对应日期的报文列表
4. **词云**: 点击标签可跳转到带该标签的报文列表
5. **表格**: 点击行可跳转到融合报告详情页

### 时间筛选
- 报文时间分布图支持切换时间范围
- 选项: 最近7天 / 最近30天 / 最近90天 / 自定义时间
- 切换后重新请求接口并更新图表

## 技术实现要点

### 1. 图表库选择
- **推荐**: Apache ECharts 5.x
- 安装: `npm install echarts`
- Vue 3 集成: 使用 `ref` + `onMounted` 初始化

### 2. 数据请求
- 使用 `Promise.all()` 并行请求多个接口
- 统一错误处理和 loading 状态管理
- 缓存策略: 5分钟内不重复请求

### 3. 响应式布局
- 使用 CSS Grid 实现自适应
- 断点: 
  - 大屏 (>1400px): 2列布局
  - 中屏 (768-1400px): 1列布局
  - 小屏 (<768px): 单列堆叠

### 4. 性能优化
- 图表懒加载（进入视口才渲染）
- 大数据集分页或虚拟滚动
- 防抖处理窗口 resize 事件
