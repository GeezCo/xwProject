## 布局说明

### 1. 页面结构
- **上下+右侧面板**: 地图回放区(flex:1) + 底部控制条(固定) | 轨迹统计(300px)
- **地图方位**: 乌方（蓝方）在左，俄方（红方）在右
- **响应式**: 小屏时统计面板收起为可展开抽屉

### 2. 选择栏
- **主目标选择**: 下拉选择要回放的目标（俄方装甲营、机步营、炮兵群等）
- **时间范围**: 选择回放起止时间
- **叠加目标**: 可同时叠加多个关联目标轨迹（不同颜色标识）
- **加载回放**: 点击后加载轨迹数据并初始化回放

### 3. 回放地图视图
- **轨迹线**: 主目标实线，叠加目标虚线/不同颜色
- **关键标记**:
  - ◉ 起点标记（集结地/出发点）
  - 🔴 当前回放位置（动态移动）
  - ● 停留点（附停留时长标注）
  - ①②③④ 关键事件编号（对应右侧事件列表）
- **态势叠加**:
  - 前线随时间变化（可显示不同日期的前线位置）
  - 蓝方(乌方,左)、红方(俄方,右)控制区域
  - 交战区域标记
  - 地形底图（道路/林地/居民点）
- **交互**:
  - 点击轨迹任意点 → 底部显示该时间点详情
  - 滚轮缩放 → 聚焦特定区段
  - 拖拽 → 平移视图

### 4. 回放控制条
- **时间轴**: 完整时间范围，标注关键事件位置
- **播放控制**: 起点/快退/播放暂停/快进/终点
- **速度控制**: 1x/2x/4x/8x/16x 倍速
- **当前位置**: 可拖拽滑块跳转到任意时间点
- **关键事件标记**: 时间轴上的标记点，可点击快速跳转

### 5. 轨迹统计面板 (右侧)
- **行军统计**: 总距离、总时间、平均/最高速度、停留次数/时长
- **速度曲线**: 速度随时间变化图
- **关键事件列表**: 按时间倒序排列，可点击跳转
- **地形统计**: 经过不同地形的比例（公路/土路/越野/林地）
- **操作**: 导出报告、对比分析

## 交互逻辑

### 1. 回放控制
```
点击播放 → 目标标记沿轨迹移动 → 时间轴同步推进 → 统计面板实时更新
拖拽时间轴 → 跳转到指定时间 → 更新地图和详情
倍速切换 → 调整回放速度
点击关键事件 → 跳转到事件时间点 → 高亮显示事件区域
```

### 2. 多目标叠加
```
添加叠加目标 → 用不同颜色绘制轨迹 → 同步回放所有目标
可切换主跟踪目标 → 地图视角跟随切换
协同关系标注 → 连线/箭头指示配合关系
```

### 3. 关键事件自动检测
```
自动检测以下事件:
- 开始/停止机动（从静止到移动或反之）
- 大角度转向 (>20°)
- 速度突变 (>50%变化)
- 长时间停留 (>30分钟)，标注可能原因（补给/集结/受阻）
- 进入/离开特定区域（前线、城镇、林地）
- 与其他目标接近/远离（疑似编队/分散）
- 疑似交战（减速+停留+附近有乌方目标）
- 疑似撤退（反向移动+速度高）
```

## 技术实现

### 1. 回放引擎
```javascript
class TrackPlaybackEngine {
  constructor(trackData, map) {
    this.track = trackData.track
    this.events = trackData.keyEvents
    this.map = map
    this.currentIndex = 0
    this.speed = 1
    this.isPlaying = false
  }

  play() {
    this.isPlaying = true
    this.animate()
  }

  pause() { this.isPlaying = false }

  setSpeed(speed) { this.speed = speed }

  animate() {
    if (!this.isPlaying || this.currentIndex >= this.track.length - 1) {
      this.isPlaying = false
      return
    }

    const current = this.track[this.currentIndex]
    const next = this.track[this.currentIndex + 1]
    const duration = (new Date(next.timestamp) - new Date(current.timestamp)) / this.speed

    const steps = Math.max(1, Math.floor(duration / 50))
    let step = 0

    const interpolate = () => {
      if (!this.isPlaying) return
      const t = step / steps
      const lon = current.lon + (next.lon - current.lon) * t
      const lat = current.lat + (next.lat - current.lat) * t
      this.updateMarker(lon, lat, this.lerpHeading(current.heading, next.heading, t))
      this.extendTrail(lon, lat)
      step++
      if (step <= steps) {
        requestAnimationFrame(interpolate)
      } else {
        this.currentIndex++
        this.checkEvents(next.timestamp)
        this.animate()
      }
    }

    requestAnimationFrame(interpolate)
  }

  seekTo(timestamp) {
    const index = this.track.findIndex(p =>
      new Date(p.timestamp) >= new Date(timestamp)
    )
    if (index >= 0) {
      this.currentIndex = index
      this.rebuildTrail(index)
      const pt = this.track[index]
      this.updateMarker(pt.lon, pt.lat, pt.heading)
    }
  }

  checkEvents(timestamp) {
    const event = this.events.find(e =>
      Math.abs(new Date(e.timestamp) - new Date(timestamp)) < 60000
    )
    if (event) this.onEvent?.(event)
  }

  lerpHeading(from, to, t) {
    let delta = ((to - from + 540) % 360) - 180
    return (from + delta * t + 360) % 360
  }
}
```

function getFrontLineAtTime(timestamp, frontLineHistory) {
  const sorted = frontLineHistory
    .filter(f => new Date(f.date) <= new Date(timestamp))
    .sort((a, b) => new Date(b.date) - new Date(a.date))
  return sorted[0]?.geoJSON || null
