## 布局说明

### 1. 页面结构
- **三栏布局**: 目标列表(240px) | 战场态势地图(flex:1) | 目标详情(340px)
- **实时更新**: WebSocket推送，15秒刷新频率（陆地场景目标变化更快）
- **响应式**: 中屏隐藏目标列表，小屏切换为单列
- **地图方位**: 乌方（蓝方）在左（西），俄方（红方）在右（东），前线居中

### 2. 目标列表 (左侧)
- **筛选器**: 按目标类型筛选
  - 装甲: 坦克、装甲车、步兵战车
  - 步兵: 机步营、摩步营、特种部队
  - 火炮: 自行火炮、火箭炮、牵引火炮
  - 防空: S-300/S-400、道尔、铠甲
  - 无人机: 察打一体、侦察型、自杀式
  - 后勤: 补给车队、弹药运输、油料运输
  - 指挥: 指挥所、通信节点
- **目标卡片**: 
  - 显示编号、类型、状态
  - 实时速度、移动方向
  - 状态指示器（机动🟢/集结🟡/交战🔴/静止⚪）
- **操作**: 点击目标卡片 → 地图居中 + 显示详情
- **排序**: 按告警优先级 → 威胁等级 → 距前线距离

### 3. 战场态势地图 (中间)
- **底图图层**:
  - 卫星影像底图
  - 地形等高线
  - 道路网络（公路、铁路、土路）
  - 河流水系
  - 居民点/城镇
  - 林地/开阔地
- **态势图层**:
  - 前线标示（竖线分隔，蓝左红右）
  - 控制区域着色（蓝方/红方/灰色地带）
  - 雷区/障碍标注
  - 防空覆盖范围（扇形）
  - 火炮射程范围（圆形）
- **目标图层**:
  - 🔵 蓝方目标（乌方，地图左侧）
  - 🔴 红方目标（俄方，地图右侧）
  - 🟡 告警目标
  - 历史轨迹线（实线）
  - 预测路线（虚线）
- **交互**:
  - 点击目标 → 显示详情
  - 右键目标 → 上下文菜单（跟踪/标记/评估威胁/生成报告）
  - 框选区域 → 区域内目标统计
  - 测距工具 → 计算两点距离/目标与前线距离

### 4. 目标详情面板 (右侧)
- **基本信息**: 编号、类型、归属（旅/师）、状态
- **实时位置**: 经纬度、速度、方向、海拔
- **运动参数**: 加速度、转向率、持续时间、行军距离
- **编成信息**: 下辖装备数量、人员规模
- **历史记录**: 首次发现、开始机动时间、最近交战
- **告警信息**: 时间戳、告警类型、描述
- **关联目标**: 协同作战关系（进攻/支援/侦察/后勤/防空）
- **操作按钮**: 查看历史轨迹、规律分析、设置告警、威胁评估

## 交互逻辑

### 1. 实时更新
```
WebSocket连接 → 接收目标位置更新 → 更新地图标记 → 更新目标列表
每15秒全量刷新 → 检查告警条件 → 触发告警通知
卫星/无人机侦察数据 → 新增/更新目标 → 更新态势图
```

### 2. 目标选择
```
点击目标卡片/地图标记 → 高亮目标 → 地图居中 → 显示详情面板
双击目标 → 跳转到轨迹历史页面
右键目标 → 显示上下文菜单（跟踪/评估/报告）
```

### 3. 轨迹显示
```
勾选"显示轨迹" → 加载最近48小时行军轨迹 → 绘制轨迹线
勾选"显示预测" → 基于历史运动+道路网络预测未来路线 → 绘制虚线
```

### 4. 告警触发
```
目标向前线快速推进 → 触发进攻预警
装甲/火炮集结超过阈值 → 触发集结告警
目标偏离常规机动路线 → 触发异常告警
后勤补给线中断/异常 → 触发后勤告警
目标进入我方火力覆盖范围 → 触发射程告警
目标消失超过阈值 → 触发失联告警（可能隐蔽/转移）
```

## 技术实现

### 1. 地图库选择
- **推荐**: Mapbox GL JS（支持3D地形、高性能渲染）
- **备选**: Leaflet + 地形插件
- **军事底图**: 支持加载WMS/WMTS军事地形图服务

### 2. 实时通信
- **WebSocket**: 推送目标位置更新
- **心跳机制**: 每10秒ping/pong保持连接
- **断线重连**: 指数退避重连策略
- **数据压缩**: 增量更新，仅推送变化字段

### 3. 性能优化
- **目标聚合**: 缩小视图时聚合同类型目标（如"装甲集群×3"）
- **轨迹简化**: Douglas-Peucker算法简化轨迹点
- **虚拟滚动**: 目标列表超过100个时启用
- **Canvas渲染**: 大量目标时使用Canvas图层
- **视野裁剪**: 仅渲染当前视野范围内的目标和轨迹

<template>

import { ref, computed, onMounted, onUnmounted } from 'vue'
import { useTrackStore } from '@/stores/track'
import { useWebSocket } from '@/composables/useWebSocket'

const trackStore = useTrackStore()
const selectedTarget = ref(null)
const showTrack = ref(true)
const showPrediction = ref(false)
const activeLayers = ref(['frontLine', 'controlZone', 'targets'])

const { connect, disconnect } = useWebSocket('ws://api/track/realtime', {
  onMessage: (data) => {
    if (data.type === 'target_update') {
      trackStore.updateTarget(data.data)
    }
  }
})

onMounted(() => {
  connect()
  trackStore.loadTargets()
  trackStore.loadFrontLine()
  trackStore.loadControlZones()
})

onUnmounted(() => disconnect())

function handleTargetSelect(target) {
  selectedTarget.value = target
  trackStore.loadTargetDetails(target.id)
}
</script>
