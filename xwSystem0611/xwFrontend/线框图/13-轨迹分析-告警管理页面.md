## 布局说明

### 1. 页面结构
- **Tab 导航**: 告警规则 | 实时告警 | 告警历史 | 告警统计
- **全宽布局**: 每个 Tab 内容占满内容区域

### 2. 告警规则类型（俄乌战场场景）
- **区域告警**: 前线逼近预警、进入我方控制区、接近关键设施
- **行为告警**: 异常时段机动、火炮阵地就位、后勤路线偏离、防空转移、撤退迹象
- **状态告警**: 目标失联（可能隐蔽/被摧毁）
- **关联告警**: 装甲集结（多单位聚集）、编队形成、协同机动

### 3. 实时告警特色
- **综合研判**: 关联多条告警自动生成态势判断
- **关联显示**: 显示告警目标的关联单位动态
- **推送指挥所**: 紧急告警可直接推送至上级指挥终端

### 4. 告警统计特色
- **按部队归属排行**: 识别活跃部队/旅级单位
- **威胁趋势**: 综合告警密度判断威胁等级变化
- **综合研判**: 自动生成战场形势判断

## 技术实现

### 1. 告警引擎（俄乌战场增强）
```javascript
class BattlefieldAlertEngine {
  evaluate(target) {
    const alerts = []

    // 前线距离告警
    if (target.distanceToFront < this.rules.frontApproach.threshold
        && this.isMovingTowardFront(target)) {
      alerts.push(this.createAlert('front_approach', target))
    }

    // 集结检测
    const nearbyTargets = this.findNearbyTargets(target, 8)
    if (nearbyTargets.length >= 5) {
      alerts.push(this.createAssemblyAlert(target, nearbyTargets))
    }

    // 火炮阵地就位
    if (target.category === 'artillery'
        && target.motion.speed === 0
        && target.terrain === 'open_field') {
      alerts.push(this.createAlert('artillery_deployed', target))
    }

    // 后勤路线偏离
    if (target.category === 'logistics') {
      const deviation = this.calculateRouteDeviation(target)
      if (deviation > 3) {
        alerts.push(this.createAlert('logistics_anomaly', target))
      }
    }

    return alerts
  }

  isMovingTowardFront(target) {
    const headingToFront = this.calculateHeadingToFront(target.position)
    const delta = Math.abs(target.motion.heading - headingToFront)
    return delta < 45 || delta > 315
  }
}
```
