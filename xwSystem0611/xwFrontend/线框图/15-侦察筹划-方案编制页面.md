## 布局说明

### 1. 页面结构
- **步骤导航**: 5步向导式流程（任务定义→区域规划→兵力编配→约束分析→方案生成）
- **渐进式展开**: 每步完成后进入下一步，可回退修改
- **视角**: 俄方筹划人员，目标是乌方区域

### 2. 步骤①: 任务定义
- 定义侦察目的、优先级、具体要求
- 关联已识别的情报空白区域
- 文本描述侦察需求

### 3. 步骤②: 区域规划
- 在地图上绘制侦察区域（支持矩形/多边形/线路）
- 标注重点目标位置
- 叠加敌方防空覆盖范围
- 评估每个区域的威胁等级

### 4. 步骤③: 兵力编配
- 展示我方可用侦察力量（无人机/侦察班/技术侦察）
- 拖拽或下拉分配力量到侦察区域
- 自动推荐编配方案（基于区域特点和可用资源）
- 显示已分配力量汇总和预计消耗

### 5. 步骤④: 约束分析
- **天气条件**: 48小时天气预报及对各侦察手段的影响评估
- **地形路况**: 地形特点、道路状况、隐蔽条件、障碍
- **敌方威胁**: 防空火力、电子战、地面反侦察能力
- **其他约束**: 自由文本录入（协同要求、时限、禁区等）

### 6. 步骤⑤: 方案生成
- **示意图**: 综合侦察方案的地图可视化
- **时间表**: 甘特图展示各行动时间节点
- **方案文书**: 标准格式的侦察方案文档（任务/兵力/时间/协同/约束）
- **操作**: 编辑文书、导出PDF、保存草稿、提交审批

<template>
  <div class="recon-plan-edit-page">
    <PageHeader :title="'侦察方案编制'" :subtitle="plan.id" />

    <StepNavigation v-model:step="currentStep" :steps="steps" />

    <MissionDefine v-if="currentStep === 1"
      v-model="plan.mission"
      :gaps="intelligenceGaps"
      @next="currentStep = 2"
    />

    <AreaPlanning v-if="currentStep === 2"
      v-model="plan.areas"
      :enemyAirDefense="airDefense"
      :frontLine="frontLine"
      @prev="currentStep = 1"
      @next="currentStep = 3"
    />

    <ForceAssignment v-if="currentStep === 3"
      v-model="plan.assignedForces"
      :areas="plan.areas"
      :availableAssets="reconAssets"
      @auto-recommend="autoRecommend"
      @prev="currentStep = 2"
      @next="currentStep = 4"
    />

    <ConstraintAnalysis v-if="currentStep === 4"
      v-model="plan.constraints"
      :weather="weatherForecast"
      :terrain="terrainData"
      :threats="threatData"
      @prev="currentStep = 3"
      @next="currentStep = 5"
    />

    <PlanGeneration v-if="currentStep === 5"
      :plan="plan"
      @prev="currentStep = 4"
      @save-draft="saveDraft"
      @submit="submitForApproval"
      @export="exportPlan"
    />
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import { useReconStore } from '@/stores/recon'
import { useRoute } from 'vue-router'

const reconStore = useReconStore()
const route = useRoute()
const currentStep = ref(1)
const plan = ref({ mission: {}, areas: [], assignedForces: {}, constraints: {}, timeline: [] })

onMounted(async () => {
  if (route.query.gap) {
    const gap = await reconStore.getGapDetail(route.query.gap)
    plan.value.mission.linkedGaps = [gap.id]
  }
  await reconStore.loadReconAssets()
  await reconStore.loadWeatherForecast()
})

async function autoRecommend() {
  const recommended = await reconStore.recommendForceAssignment(plan.value.areas, plan.value.constraints)
  plan.value.assignedForces = recommended
}

async function submitForApproval() {
  await reconStore.submitPlan(plan.value)
}
</script>
