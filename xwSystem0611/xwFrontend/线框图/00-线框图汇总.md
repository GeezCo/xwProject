## 核心功能流程

### 属性抽取流程
```
用户点击列表项
      │
      ▼
右侧显示详情面板
      │
      ▼
点击"属性抽取"按钮
      │
      ▼
前端调用 POST /extraction/extract?originTextId=xxx
      │
      ▼
后端转发到算法服务(端口5001)
      │
      ▼
算法服务执行层次化抽取
  ├─ 第一层: 规则筛选
  └─ 第二层: LLM精确抽取
      │
      ▼
结果返回并保存到数据库
      │
      ▼
前端展示抽取事件列表
```

### 报文融合流程
```
【方式1: 手动选择报文】
用户在搜索结果页勾选多篇报文(最少2篇，最多10篇)
      │
      ▼
点击"报文融合"按钮
      │
      ▼
跳转到报文融合结果页面(fusionResult.vue)
      │
      ▼
前端调用 POST /api/fusion/create (fusionType: "manual")
      │
      ▼
后端调用LLM对多篇报文进行融合分析
      │
      ▼
AI生成结构化融合报告
  ├─ 报告标题(自动生成)
  ├─ 报告摘要
  ├─ 事件时间线(按时间排序)
  ├─ 详细内容(分章节)
  ├─ 关键实体(人物/组织/部队/军舰/火炮/战机/武器，共7类)
  └─ 综合标签(标签聚合)
      │
      ▼
用户操作: 保存/导出PDF/导出Word/重新融合/编辑

【方式2: 目标名称搜索】
用户在融合结果页面选择"目标名称搜索"模式
      │
      ▼
输入目标名称(如"第七舰队"、"B-52轰炸机")
      │
      ▼
前端调用 POST /api/fusion/searchByTarget
      │
      ▼
后端搜索包含该目标的所有报文(最多10篇)
      │
      ▼
自动调用 POST /api/fusion/create (fusionType: "target")
      │
      ▼
后端调用LLM对目标相关报文进行融合分析
      │
      ▼
AI生成针对该目标的专题融合报告
      │
      ▼
用户操作: 保存/导出PDF/导出Word/重新搜索/编辑
```
