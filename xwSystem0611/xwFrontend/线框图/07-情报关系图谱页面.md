## 布局说明

### 1. 页面结构
- **三栏布局**: 筛选面板(220px) | 图谱主视图(flex:1) | 详情面板(340px)
- **响应式**: 中屏隐藏筛选面板，小屏切换为单列

### 2. 筛选面板 (左侧)
- **实体类型筛选**: 复选框列表，显示每种类型的数量
- **关系类型筛选**: 复选框列表，显示每种关系的数量
- **时间范围筛选**: 滑块选择器，显示每月数据密度
- **操作按钮**: 应用筛选、重置筛选

### 3. 图谱主视图 (中间)
- **交互式力导向图**: 
  - 节点: 不同颜色代表不同实体类型
  - 边: 不同样式代表不同关系类型
  - 节点大小: 根据出现频次动态调整
  - 可拖拽、缩放、平移
- **图例说明**: 底部显示节点和关系的视觉编码
- **底部控制栏**: 缩放、重置、展开/收起节点、状态显示

### 4. 详情面板 (右侧)
- **实体信息**: 类型、名称、统计数据
- **关联实体列表**: 显示直接关联的其他实体
- **相关报文列表**: 显示提及该实体的所有报文
- **操作按钮**: 查看时间线、生成报告、添加标注

## 交互逻辑

### 1. 节点交互
```
点击节点 → 高亮该节点及其直接关联 → 右侧显示详情
双击节点 → 展开该节点的二度关联
右键节点 → 显示上下文菜单(隐藏/固定/查看详情/生成子图)
悬停节点 → 显示tooltip(名称、类型、出现次数)
```

### 2. 筛选交互
```
勾选实体类型 → 实时更新图谱显示
勾选关系类型 → 实时更新边的显示
调整时间范围 → 过滤指定时间段的数据
搜索实体 → 高亮匹配节点并居中显示
```

### 3. 布局算法
```
力导向布局 (默认) → 自动优化节点位置
层次布局 → 按层级关系排列
径向布局 → 以选中节点为中心辐射
网格布局 → 规则网格排列
```

### 4. 导出功能
```
导出图片 → PNG/SVG格式
导出数据 → JSON/CSV格式
生成报告 → 基于选中节点生成关系分析报告
```

## 技术实现

### 1. 图谱渲染库
- **推荐**: Cytoscape.js (功能强大，支持复杂布局)
- **备选**: D3.js (灵活性高，学习曲线陡)
- **备选**: G6 (AntV，中文文档友好)

### 2. 性能优化
- **大规模数据**: 
  - 节点数 > 500: 启用虚拟化渲染
  - 节点数 > 1000: 分批加载，按需展开
- **布局计算**: 使用 Web Worker 避免阻塞主线程
- **缓存策略**: 缓存已加载的子图数据

import { ref, computed, onMounted } from 'vue'

import Cytoscape from 'cytoscape'

const selectedNode = ref(null)
const layoutAlgorithm = ref('force')

onMounted(async () => {
  await graphStore.loadGraph()

function initCytoscape() {
  const cy = Cytoscape({
    container: document.getElementById('graph-canvas'),
    elements: {
      nodes: graphStore.nodes,
      edges: graphStore.edges
    },
    style: [
      {
        selector: 'node[type="人物"]',
