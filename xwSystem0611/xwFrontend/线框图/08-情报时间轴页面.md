## 布局说明

### 1. 页面结构
- **三栏布局**: 筛选面板(220px) | 时间轴主视图(flex:1) | 详情面板(340px)
- **响应式**: 中屏隐藏筛选面板，小屏切换为单列

### 2. 筛选面板 (左侧)
- **事件类型筛选**: 复选框列表，显示每种类型的数量
- **地区筛选**: 复选框列表，按地理区域分类
- **重要性筛选**: 高/中/低三个等级
- **数据来源筛选**: 新闻/社交媒体/官方文件
- **操作按钮**: 应用筛选、重置筛选

### 3. 时间轴主视图 (中间)
- **时间刻度尺**: 顶部显示时间刻度，可拖拽滚动
- **事件节点**: 
  - 垂直排列，按时间顺序从上到下
  - 不同颜色代表不同事件类型
  - 节点大小代表重要性等级
  - 连线表示事件之间的因果关系
- **交互功能**:
  - 点击节点: 右侧显示详情
  - 悬停节点: 显示简要信息
  - 拖拽时间轴: 查看不同时间段
  - 缩放: 调整时间粒度(天/周/月)
- **图例说明**: 底部显示颜色和符号含义

### 4. 详情面板 (右侧)
- **事件信息**: 标题、时间、地点、类型、重要性
- **事件摘要**: AI生成的简要描述
- **关键实体**: 提取的人物、组织、地点等
- **相关报文列表**: 链接到原始报文
- **关联事件**: 显示因果相关的其他事件
- **操作按钮**: 查看关系图谱、生成分析报告、添加到工作区

### 5. 底部控制栏
- **播放控制**: 前进/后退/快进/快退按钮
- **播放速度**: 滑块调整动画速度
- **自动播放**: 按时间顺序自动展示事件
- **状态显示**: 当前时间、显示事件数、选中事件

## 交互逻辑

### 1. 时间轴交互
```
点击事件节点 → 右侧显示详情 → 高亮关联事件
双击事件节点 → 跳转到关系图谱页面，以该事件为中心
右键事件节点 → 显示上下文菜单(查看报文/添加标注/隐藏事件)
悬停事件节点 → 显示tooltip(标题、时间、类型)
拖拽时间轴 → 滚动查看不同时间段
滚轮缩放 → 调整时间粒度(天/周/月视图)
```

### 2. 筛选交互
```
勾选事件类型 → 实时更新时间轴显示
勾选地区 → 过滤指定地区的事件
调整重要性 → 只显示指定等级的事件
选择数据来源 → 过滤指定来源的事件
```

### 3. 播放动画
```
点击播放按钮 → 从当前时间开始自动播放
播放过程中 → 逐个高亮事件节点，右侧同步显示详情
调整速度 → 改变播放速度(0.5x - 2x)
暂停播放 → 停留在当前事件
```

### 4. 时间范围切换
```
最近7天 → 显示近一周的事件
最近30天 → 显示近一月的事件
最近3个月 → 显示近三月的事件
自定义范围 → 选择起止日期
```

### 修改 Home.vue 菜单栏
```vue
<div class="menu-bar">
  <div class="menu-item" :class="{ active: currentRoute === 'SearchResult' }" @click="navigate('SearchResult')">
    动态信息
  </div>
  <div class="menu-item" :class="{ active: currentRoute === 'Dashboard' }" @click="navigate('Dashboard')">
    数据看板
  </div>
  <div class="menu-item" :class="{ active: currentRoute === 'IntelligenceGraph' }" @click="navigate('IntelligenceGraph')">
    关系图谱
  </div>
  <div class="menu-item" :class="{ active: currentRoute === 'IntelligenceTimeline' }" @click="navigate('IntelligenceTimeline')">
    时间轴
  </div>
  <div class="menu-spacer"></div>
</div>
```

### 路由配置
```javascript
{
  path: '/Home',
  component: Home,
  children: [
    { path: 'searchResult', name: 'SearchResult', component: SearchResult },
    { path: 'fusionResult', name: 'FusionResult', component: FusionResult },
    { path: 'dashboard', name: 'Dashboard', component: Dashboard },
    { path: 'intelligenceGraph', name: 'IntelligenceGraph', component: IntelligenceGraph },
    { path: 'intelligenceTimeline', name: 'IntelligenceTimeline', component: IntelligenceTimeline }  // 新增
  ]
}
```
