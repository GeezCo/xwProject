### 技术实现
```
<html data-theme="dark">  ←→  <html data-theme="light">

CSS变量自动切换:
  :root, [data-theme="dark"]  { --color-primary: #009afb; ... }
  [data-theme="light"]        { --color-primary: #1a73e8; ... }

持久化: localStorage.setItem('theme', 'dark' | 'light')
```

### 设计原则
1. **布局完全一致** — 暗色/亮色只替换色值，不改变任何布局和间距
2. **暗色用渐变，亮色用纯色** — 暗色主题按钮/菜单用蓝色渐变有科技感；亮色用纯色简洁
3. **暗色无阴影，亮色有阴影** — 暗色靠边框区分层级；亮色靠阴影
4. **暗色用背景图，亮色不用** — 亮色模式下 loginBg/loginT/headerBgAn/bg 全部隐藏
5. **标题色降饱和** — 亮色背景下 #fba700→#d48806 避免刺眼
6. **正文色反转** — 暗底浅字(#cdcdcd) ↔ 亮底深字(#333333)
