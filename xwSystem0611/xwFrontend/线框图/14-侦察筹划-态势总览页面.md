## 布局说明

### 1. 页面结构
- **地图+右侧面板**: 战场态势图(flex:1) | 态势摘要面板(320px)
- **底部信息栏**: 固定高度，显示我方侦察资源和任务状态
- **地图方位**: 敌方(乌方/蓝方)在左，我方(俄方/红方)在右
- **视角**: 站在俄方，对乌方区域进行侦察筹划

### 2. 战场态势全图
- **我方部署（右侧红方）**: 我方旅/营级单位位置和状态
- **已知敌情（左侧蓝方）**: 已掌握的乌方防御阵地、兵力部署
  - 标注信息确定度（如"约2个营?" "??? 兵力不明"）
- **情报空白区（虚线框）**: 我方对敌方缺乏情报的区域
  - 标注最后侦察时间、推测内容
  - 这些区域是侦察筹划的核心需求来源
- **敌方防空覆盖**: 标注乌方S-300、山毛榉等防空火力范围（影响无人机侦察）

### 3. 态势摘要面板 (右侧)
- **敌情概要**: 已知乌方单位数量、防御纵深判断（标注不确定项）
- **情报空白区列表**: 需要侦察的区域，按优先级排列
- **约束条件**:
  - 天气: 影响光学侦察和地面机动
  - 敌方防空: 限制无人机高度和航线
  - 敌方电子战: 影响通信和GPS导航
- **操作入口**: 发起侦察筹划、查看历史方案

### 4. 底部信息栏
- **我方侦察力量**: 猎户座(察打)、海雕-10(侦察)、侦察班、电子侦察、卫星
- **进行中任务**: 正在执行/返航的侦察任务
- **待筹划任务**: 等待审批和草稿状态
- **敌区侦察覆盖率**: 对乌方区域的情报覆盖比例

## 交互逻辑

### 1. 态势浏览
```
选择战区 → 地图聚焦对应区域 → 加载已知敌情+情报空白
切换时间 → 查看不同时间态势快照
图层控制 → 开关敌方防空/电子战/雷区/地形等图层
```

### 2. 情报空白区交互
```
点击情报空白区 → 高亮范围 → 显示详细信息(推测内容/最后侦察/优先级)
右键情报空白区 → 快速发起针对该区域的侦察筹划
双击 → 放大查看该区域详细地形
```

### 3. 发起筹划
```
点击"发起侦察筹划" → 跳转方案编制页
   → 自动带入当前敌情和我方态势
   → 自动带入天气/防空/电子战约束
   → 推荐侦察区域(基于情报空白优先级)
   → 推荐侦察力量(基于可用资源和威胁等级)
```
