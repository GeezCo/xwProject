### 1. Tab 1 - 方案评估
- **综合评分**: 可行性、风险、资源效率、时效性
- **风险分析**: 按高/中/低风险分级，每项附建议措施
- **可行性分析**: 有利因素和不利因素，附应对方案
- **方案对比**: 表格对比多个备选方案的关键指标

### 2. Tab 2 - 审批流转
- **流转状态**: 可视化审批流程节点（编制→科长→处长→首长→下达）
- **审批记录**: 每个环节的审批意见和建议
- **待审批列表**: 当前等待审批的方案

### 3. Tab 3 - 执行跟踪
- **执行态势图**: 实时显示侦察力量位置和完成区域（乌方左/俄方右）
- **执行进度**: 分阶段进度条（渗透→空中侦察→情报分析→撤收）
- **关键时间节点**: 计划与实际对比
- **实时事件流**: 滚动显示侦察回传信息和AI识别结果
- **紧急操作**: 暂停、调整、紧急撤收

### 4. Tab 4 - 成果汇总
- **成果概要**: 图像数量、发现目标数、报告数、完成率
- **成果标注图**: 在地图上标注所有发现目标（新发现/更新/确认/排除）
- **目标清单**: 按新发现和更新分类，含位置、来源、可信度
- **情报空白填补**: 对比侦察前后情报覆盖率变化
- **操作**: 导出报告、提交成果、更新态势、发起后续侦察

        </div>

      </a-tab-pane>

          <TargetList :targets="results.targets" @view="viewTarget" />

      </a-tab-pane>
    </a-tabs>

import { ref, onMounted, onUnmounted } from 'vue'
import { useReconStore } from '@/stores/recon'
import { useWebSocket } from '@/composables/useWebSocket'

const reconStore = useReconStore()
const activeTab = ref('evaluation')
const selectedPlan = ref(null)

const { connect, disconnect } = useWebSocket('ws://api/recon/execution', {
  onMessage: (data) => {
    if (data.type === 'execution_update') {
      reconStore.updateExecution(data.data)
    }
  }
})

onMounted(() => {
  connect()

onUnmounted(() => disconnect())
