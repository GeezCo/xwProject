交互流程:
1. 进入页面 → 自动加载分类目录和报文列表（按 sid 倒序，默认全部）
2. 点击信息来源目录 → 筛选对应信息来源分类数据
3. 点击报文模态目录 → 筛选对应报文类型数据（文字报/图文报/声像报）
4. 顶部「数据来源」下拉框 → 单选一级分类快速筛选
5. 点击列表项 → 右侧显示详情（图文报自动展示图片网格）
6. 点击图片 → 放大预览（支持缩放）
7. 点击"属性抽取"按钮 → 显示名言轮播等待界面 → 抽取完成后显示结果
8. 勾选多篇报文 → 列表顶部显示已选择数量 → 点击"报文融合"按钮 → 弹出报文融合结果页面
9. 输入目标名称（如"第七舰队"）点击搜索 → 在已抽取报文中检索相关报文用于融合

标签数据结构:
```json
{
  "extractionResult": {
    "labels": ["人物", "组织", "飞机", "导弹"],  // 整体标签（所有事件标签去重，共11种：船只/飞机/无人机/导弹/雷达/火炮/坦克装甲车辆/军事基地/部队番号/人物/组织）
    "entities": {                                // 实体按11类中文key归类（顶层汇总，仅显示非空类别）
      "人物": ["拜登"],
      "组织": ["白宫"],
      "飞机": ["B-52轰炸机"],
      "部队番号": ["第七舰队"],
      "导弹": ["战略导弹"]
    },
    "events": [
      {
        "time": "2024年3月15日",
        "location": ["白宫"],
        "subject": ["拜登", "白宫"],
        "action": "签署行政命令",
        "labels": ["人物", "组织"],
        "original_text": "拜登在白宫签署行政命令..."
      }
    ]
  }
}
```

事件展示说明:
- 不显示原始subject字段，改为按实体分类展示
- 利用顶层entities对象匹配每个事件的subject项，按分类标签分行显示（仅展示非空分类）
- 每个事件末尾显示original_text（虚线分隔、斜体）
