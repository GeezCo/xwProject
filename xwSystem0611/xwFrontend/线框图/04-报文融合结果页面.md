融合流程说明:
┌─────────────────────────────────────────────────────────────────────────┐
│ 步骤                  │ 操作                                               │
├─────────────────────────────────────────────────────────────────────────┤
│ 【方式1: 手动选择报文】                                                   │
│ 1. 选择报文           │ 在搜索结果页面勾选多篇报文（最少2篇，最多10篇）     │
│ 2. 点击融合           │ 点击"报文融合"按钮，进入融合结果页面               │
│ 3. AI处理             │ 系统调用LLM对报文内容进行融合分析                   │
│ 4. 生成报告           │ AI生成结构化的融合报告                              │
│ 5. 用户操作           │ 用户可保存、导出、重新融合或编辑报告                │
│                                                                           │
│ 【方式2: 目标名称搜索】                                                   │
│ 1. 选择融合方式       │ 在融合页面顶部选择"目标名称搜索"                   │
│ 2. 输入目标名称       │ 在搜索框输入目标（如"第七舰队"、"B-52轰炸机"）     │
│ 3. 点击搜索           │ 系统自动搜索包含该目标的所有报文                   │
│ 4. 自动融合           │ 系统自动选择相关报文并调用LLM融合                   │
│ 5. 生成报告           │ AI生成针对该目标的专题融合报告                      │
│ 6. 用户操作           │ 用户可保存、导出、重新搜索或编辑报告                │
└─────────────────────────────────────────────────────────────────────────┘

API接口设计:
```
POST /api/fusion/create
请求参数:
{
  "reports": [                   // 选中的报文数据列表
    {
      "id": 9751,                // 报文ID
      "title": "报文标题",
      "content": "报文正文内容",
      "times": "2024-03-15",
      "type": "军事演习",
      "extractionResult": {      // 该报文的抽取结果
        "events": [...],
        "entities": {...},
        "labels": [...]
      }
    }
  ],
  "fusionType": "manual",        // 融合类型：manual(手动选择) / target(目标搜索)
  "targetName": "",              // 目标名称（仅target模式需要）
  "customTitle": ""              // 自定义标题（可选）
}

POST /api/fusion/searchByTarget
请求参数:
{
  "targetName": "第七舰队",      // 目标名称
  "maxReports": 10               // 最多返回报文数（默认10）
}

响应:
{
  "code": 200,
  "message": "success",
  "data": {
    "reports": [                 // 匹配的报文列表
      {
        "id": 9751,
        "title": "报文标题",
        "content": "报文正文内容",
        "times": "2024-03-15",
        "type": "军事演习",
        "extractionResult": {...}
      }
    ],
    "total": 5                   // 匹配的报文总数
  }
}

响应:
{
  "fusionId": 100,               // 融合报告ID
  "title": "北约军事动态综合报告",
  "summary": "本报告综合了...",
  "timeline": [...],             // 事件时间线
  "content": "...",              // 详细内容(Markdown格式)
  "entities": {                  // 关键实体（11类，仅返回非空类别）
    "人物": ["拜登"],
    "组织": ["美军", "北约"],
    "部队番号": ["第七舰队"],
    "飞机": ["B-52轰炸机"],
    "导弹": ["战略导弹"]
  },
  "labels": ["人物", "组织", "部队番号", "飞机", "导弹"],
  "createTime": "2024-04-15 14:30:25",
  "modelUsed": "Qwen/Qwen3.5-35B-A3B-w8a8-mtp"
}
```
