# 数据生成

基于 LLM 的文本仿写生成器，读取样例文本，通过候选池随机化批量生成相似风格的报文数据。

## 文件说明

| 文件 | 说明 |
|------|------|
| data_generator.py | 主程序：DataGenerator 类，支持并行生成、占位符替换、条目数量偏离 |
| candidate_pools.py | 候选池配置：时间、地点、人物、组织、装备、行为六类候选池 |
| split_events.py | 辅助脚本：将多事件文本拆分为单事件文件 |
| requirements.txt | Python 依赖清单（openai） |
| LLM_api.txt | LLM API 配置参考 |
| CLAUDE.md | Claude Code 项目指导文件 |

## 子目录

| 目录 | 说明 |
|------|------|
| 样例/ | 仿写样例模板（1 篇） |
| 样例-20260406/ | 2026-04-06 版样例模板（7 篇，编号 0-6） |
| 输出/ | 生成的报文输出（按样例编号分子目录） |
| 输出-20260406/ | 2026-04-06 批次生成输出 |
| 测试输出/ | 测试生成的输出 |
| 服务器说明/ | 服务器相关配置文件 |

## 使用方式

```bash
python data_generator.py <样例目录> <输出目录> --total 100 --parallel 5
```
