# 数据抽取-带标签

**当前生产版**事件抽取算法，在旧版基础上增加了 RexUniNlu 多标签分类功能，为每个事件和实体自动标注类别标签。

## 文件说明

| 文件 | 说明 |
|------|------|
| hierarchical_extractor.py | 层次化事件抽取主入口：规则筛选 → LLM 精确抽取 → 标签分类 |
| llm_event_extractor.py | LLM 事件抽取底层，含 SYSTEM_PROMPT 定义抽取格式要求 |
| classify_client.py | RexUniNlu 分类客户端，对事件主体进行多标签分类，自动拼接"其他"作为兜底并过滤 |
| element_extractor.py | 纯规则抽取（保留兼容） |
| README.md | 模块说明文档，含三种抽取方式对比 |
| requirements.txt | Python 依赖清单 |
| sample.txt | 示例输入文本 |
| sample_extracted.json | 示例抽取结果 |
| test_mixed.txt | 混合测试文本 |
| 测试用例1.txt | 测试用例 |
| LLM_api.txt | LLM API 配置参考 |
| 软件设计说明文档.docx | 设计文档 |

## 子目录

| 目录 | 说明 |
|------|------|
| 测试数据/ | 测试用的输入文本文件（10 篇军事报文） |
| 测试数据/results/ | 测试抽取输出的 JSON 结果 |
