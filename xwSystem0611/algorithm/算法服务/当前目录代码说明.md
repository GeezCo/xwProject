# 算法服务

Flask REST API 服务，为后端 Spring Boot 提供事件抽取和报文融合的 HTTP 接口，运行在端口 5001。

## 文件说明

| 文件 | 说明 |
|------|------|
| app.py | Flask 主程序，注册路由、导入抽取/融合模块、启动服务 |
| requirements.txt | Python 依赖清单（flask, flask-cors, openai, requests） |

## 接口列表

| 路径 | 方法 | 功能 |
|------|------|------|
| /health | GET | 健康检查 |
| /extract | POST | 完整事件抽取（层次化 + 标签分类） |
| /extract/simple | POST | 简化事件抽取（无持久化） |
| /fusion/create | POST | 多报文融合生成报告 |

## 启动方式

```bash
pip install -r requirements.txt
python app.py
```
