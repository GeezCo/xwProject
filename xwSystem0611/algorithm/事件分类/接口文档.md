## 接口详细说明

### 1. 健康检查

检查服务运行状态和模型加载情况。

**请求**
```
GET /health
```

**响应参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| status | string | 服务状态，"ok"表示正常 |
| model | string | 加载的模型名称 |

**响应示例**
```json
{
    "status": "ok",
    "model": "rex-uninlu"
}
```

**请求示例**
```bash
curl -X GET http://localhost:8514/health
```

---

### 2. 单标签分类

对文本进行单标签分类，返回最匹配的一个标签。

**请求**
```
POST /api/v1/classify
Content-Type: application/json
```

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| text | string | 是 | 待分类文本，最大长度建议不超过512字符 |
| labels | array[string] | 是 | 分类标签列表，至少包含1个标签 |

**响应参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| label | string | 预测的分类标签 |
| confidence | float | 置信度，范围0.0-1.0 |

**请求示例**
```bash
curl -X POST http://localhost:8514/api/v1/classify \
  -H 'Content-Type: application/json' \
  -d '{
    "text": "歼-20隐身战斗机具备优异的超音速巡航能力",
    "labels": ["人物", "组织", "部队", "军舰", "火炮", "战机"]
  }'
```

**响应示例**
```json
{
    "label": "战机",
    "confidence": 1.0
}
```

**Python调用示例**
```python
import requests

response = requests.post(
    "http://localhost:8514/api/v1/classify",
    json={
        "text": "张伟同志担任某集团军装甲师师长",
        "labels": ["人物", "组织", "部队", "军舰", "火炮", "战机"]
    }
)
result = response.json()
print(f"分类结果: {result['label']}, 置信度: {result['confidence']}")
```

### 3. 多标签分类

对文本进行多标签分类，返回多个匹配标签及其置信度。

**请求**
```
POST /api/v1/classify/multi
Content-Type: application/json
```

**请求参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| text | string | 是 | - | 待分类文本 |
| labels | array[string] | 是 | - | 分类标签列表 |
| top_k | integer | 否 | 3 | 返回top k个最匹配的标签 |

**响应参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| labels | array[string] | 预测的标签列表，按置信度降序排列 |
| scores | array[float] | 对应标签的置信度列表 |

**请求示例**
```bash
curl -X POST http://localhost:8514/api/v1/classify/multi \
  -H 'Content-Type: application/json' \
  -d '{
    "text": "今天阳光明媚，心情特别愉快，适合外出游玩",
    "labels": ["情感", "财经", "科技", "生活", "新闻"],
    "top_k": 3
  }'
```

**响应示例**
```json
{
    "labels": ["情感", "生活"],
    "scores": [0.92, 0.78]
}
```

**Python调用示例**
```python
import requests

response = requests.post(
    "http://localhost:8514/api/v1/classify/multi",
    json={
        "text": "中国科学院物理研究所取得重要进展",
        "labels": ["人物", "组织", "部队", "军舰", "火炮", "战机"],
        "top_k": 2
    }
)
result = response.json()
for label, score in zip(result['labels'], result['scores']):
    print(f"{label}: {score:.2f}")
```

### 4. 批量分类

对多个文本进行批量单标签分类。

**请求**
```
POST /api/v1/classify/batch
Content-Type: application/json
```

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| texts | array[string] | 是 | 待分类文本列表，建议单次不超过100条 |
| labels | array[string] | 是 | 分类标签列表 |

**响应参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| results | array[object] | 分类结果列表 |

**results数组元素**

| 参数 | 类型 | 说明 |
|------|------|------|
| text | string | 原始文本 |
| label | string | 预测的分类标签 |

**请求示例**
```bash
curl -X POST http://localhost:8514/api/v1/classify/batch \
  -H 'Content-Type: application/json' \
  -d '{
    "texts": [
      "张伟同志担任某集团军装甲师师长",
      "中国科学院物理研究所取得重要进展",
      "歼-20隐身战斗机具备优异性能"
    ],
    "labels": ["人物", "组织", "部队", "军舰", "火炮", "战机"]
  }'
```

**响应示例**
```json
{
    "results": [
        {"text": "张伟同志担任某集团军装甲师师长", "label": "人物"},
        {"text": "中国科学院物理研究所取得重要进展", "label": "组织"},
        {"text": "歼-20隐身战斗机具备优异性能", "label": "战机"}
    ]
}
```

**Python调用示例**
```python
import requests

texts = [
    "辽宁号航空母舰是中国第一艘服役的航母",
    "PLZ-05式155毫米自行榴弹炮射程达到50公里",
    "运-20大型运输机提升了战略投送能力"
]

response = requests.post(
    "http://localhost:8514/api/v1/classify/batch",
    json={
        "texts": texts,
        "labels": ["人物", "组织", "部队", "军舰", "火炮", "战机"]
    }
)

for item in response.json()['results']:
    print(f"{item['text'][:20]}... -> {item['label']}")
```

## 使用建议

### 1. 标签设计建议
- 标签之间应互斥，避免语义重叠
- 标签名称简洁明确，建议2-4个汉字
- 标签数量建议不超过10个

### 2. 文本预处理建议
- 去除无关的特殊字符和空白
- 保持文本语义完整
- 避免过长的文本输入

### 3. 性能优化建议
- 批量请求优先使用 `/api/v1/classify/batch`
- 合理设置 `top_k` 参数，避免返回过多结果
- 对于高并发场景，建议实现客户端连接池

## 完整调用示例

### Python完整示例

```python
import requests
from typing import List, Dict

class RexNLUClient:
    """RexNLU分类服务客户端"""
    
    def __init__(self, base_url: str = "http://localhost:8514"):
        self.base_url = base_url
    
    def health_check(self) -> Dict:
        """健康检查"""
        response = requests.get(f"{self.base_url}/health")
        return response.json()
    
    def classify(self, text: str, labels: List[str]) -> Dict:
        """单标签分类"""
        response = requests.post(
            f"{self.base_url}/api/v1/classify",
            json={"text": text, "labels": labels}
        )
        return response.json()
    
    def classify_multi(self, text: str, labels: List[str], top_k: int = 3) -> Dict:
        """多标签分类"""
        response = requests.post(
            f"{self.base_url}/api/v1/classify/multi",
            json={"text": text, "labels": labels, "top_k": top_k}
        )
        return response.json()
    
    def classify_batch(self, texts: List[str], labels: List[str]) -> Dict:
        """批量分类"""
        response = requests.post(
            f"{self.base_url}/api/v1/classify/batch",
            json={"texts": texts, "labels": labels}
        )
        return response.json()


# 使用示例
if __name__ == "__main__":
    client = RexNLUClient()
    
    # 健康检查
    print("服务状态:", client.health_check())
    
    # 单标签分类
    result = client.classify(
        text="歼-20隐身战斗机具备优异性能",
        labels=["人物", "组织", "部队", "军舰", "火炮", "战机"]
    )
    print(f"分类结果: {result}")
    
    # 多标签分类
    result = client.classify_multi(
        text="今天心情很好，天气也不错",
        labels=["情感", "财经", "科技", "生活", "新闻"],
        top_k=2
    )
    print(f"多标签结果: {result}")
    
    # 批量分类
    result = client.classify_batch(
        texts=["张伟担任师长", "华为发布新系统", "歼-20战机服役"],
        labels=["人物", "组织", "部队", "军舰", "火炮", "战机"]
    )
    print(f"批量结果: {result}")
```
