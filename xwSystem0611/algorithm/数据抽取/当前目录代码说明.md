# 数据抽取

旧版事件抽取算法（不含标签分类），保留作为参考。生产环境使用 `数据抽取-带标签/` 目录。

## 文件说明

| 文件 | 说明 |
|------|------|
| hierarchical_extractor.py | 层次化事件抽取（主入口）：规则筛选 + LLM 精确抽取，节省约 50% API 调用 |
| llm_event_extractor.py | LLM 事件抽取底层实现，调用大模型进行结构化信息提取 |
| element_extractor.py | 纯规则抽取（正则 + 分词），无需 LLM，速度快但精度有限 |
| classify_client.py | RexUniNlu 分类服务客户端（旧版，无"其他"兜底过滤） |
| config.json | 运行时配置：LLM 模型、API 地址、实体标签列表 |
| config.example.json | 配置文件模板 |
| requirements.txt | Python 依赖清单 |
| sample.txt | 示例输入文本 |
| sample_extracted.json | 示例抽取结果 |
| test_mixed.txt | 混合测试文本 |
| test_mixed_hierarchical_events.json | 混合文本的层次化抽取结果 |
| 测试用例1.txt | 测试用例 |
| LLM_api.txt | LLM API 配置参考 |
| 软件设计说明文档.docx | 设计文档 |

## 子目录

| 目录 | 说明 |
|------|------|
| 测试数据/ | 测试用的输入文本文件 |
| 测试数据/results/ | 测试抽取输出的 JSON 结果 |
